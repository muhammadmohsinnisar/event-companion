# Event Companion App

🎉 An open-source Android app for managing events and tickets!

---

## Disclaimer
This is a prototype demonstrating the basic structure of an Event Companion app.
Currently it uses a mocked API for testing purposes: https://github.com/muhammadmohsinnisar/eventcompanion-mock-api.
Future roadmap includes adding backend integration, real event QR flow, multi-event management, and user accounts.

## Features

✅ Scan event-provided QR codes to receive your ticket  
✅ View schedule of event sessions  
✅ View list of event speakers  
✅ Offline-friendly: schedule is cached locally  
✅ Modern UI built with Jetpack Compose  
✅ Easy to extend for multiple events and more features!

---

## How It Works

1️⃣ Organizer provides QR code for ticket  
2️⃣ User scans QR code using built-in scanner  
3️⃣ App displays ticket and event details  
4️⃣ User can browse schedule and speakers  
5️⃣ Prototype version now — roadmap supports multi-event handling
